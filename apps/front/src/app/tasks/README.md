# Frontend: Tasks

**Purpose:** Task list and task detail UI for user to-dos with multiple view modes.

**Key responsibilities:**
- Create/edit tasks, reminders and recurrence
- Link tasks to leads/deals/contacts
- Multiple view modes: List, Board (Kanban), Calendar
- Drag & drop task management on Board view

**View Modes:**
- **List View** (`/tasks`) - traditional table view with filters and pagination
- **Board View** (`/tasks/board`) - Kanban-style board with drag & drop
- **Calendar View** (`/tasks/calendar`) - calendar-based task scheduling

**Notes:** Support quick-create and calendar integrations.

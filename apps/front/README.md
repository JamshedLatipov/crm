# CRM Frontend

This is the main frontend application for the CRM system, built with [Angular](https://angular.io/).

## Overview

The frontend provides a comprehensive user interface for managing contacts, deals, tasks, and communications. It includes a softphone implementation for handling calls directly within the browser.

## Key Technologies

- **Framework:** Angular
- **UI Library:** Angular Material
- **Styling:** Tailwind CSS & SCSS
- **Charts:** Chart.js / ng2-charts
- **Telephony:** JSSIP (WebRTC)

## Getting Started

### Running the Application

To start the frontend in development mode:

```bash
nx serve front
```

The application will be available at `http://localhost:4200`.

### Testing

Run unit tests:

```bash
nx test front
```

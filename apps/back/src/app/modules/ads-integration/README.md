# Модуль Рекламных Интеграций (Ads Integration)

## Описание
Модуль для интеграции с рекламными платформами (Facebook Ads, Google Ads и др.) для автоматического сбора лидов и аналитики расходов.

## Компоненты
- `AdsIntegrationService`: Управление подключениями.
- `AdsSyncService`: Фоновая синхронизация данных.
- `Adapters`: Адаптеры для конкретных внешних API.

## Функциональность
- Подключение рекламных аккаунтов (OAuth).
- Загрузка лидов (Lead Gen Forms).
- Сопоставление расходов с полученными сделками (ROI).

# Модуль Активности Пользователей (User Activity)

## Описание
Модуль аудита и логирования действий пользователей в системе (Audit Log).

## Функциональность
- Запись событий (вход в систему, изменение данных, экспорт отчетов).
- Просмотр истории действий конкретного пользователя.
- Анализ продуктивности сотрудников.

## Компоненты
- `UserActivityService`: Сервис для записи логов.
- `UserActivityEntity`: Структура хранилища логов.

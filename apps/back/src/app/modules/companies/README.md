# Модуль Компаний (Companies Module)

## Описание
Модуль предназначен для управления базой юридических лиц и организаций. Позволяет хранить реквизиты, связывать компании с контактами и сделками.

## Основные компоненты

### Сущности (Entities)
- `CompanyEntity` (в папке `entities`): Основная сущность компании. Хранит название, адрес, отрасль, реквизиты.

### Сервисы (Services)
- `CompaniesService`: Бизнес-логика управления компаниями.

### Контроллеры (Controllers)
- `CompaniesController`: REST API для управления компаниями.

## Функциональность
- Создание и редактирование профилей компаний.
- Привязка сотрудников (контактов) к компаниям.
- Отслеживание истории взаимодействия на уровне компании.

## API Endpoints (Ожидаемые)
- `GET /api/companies`
- `POST /api/companies`
- `GET /api/companies/:id`
- `PATCH /api/companies/:id`
- `DELETE /api/companies/:id`

# Модуль Контакт-центра (Contact Center Module)

## Описание
Модуль обеспечивает функционал реального времени для операторов контакт-центра. Использует WebSockets для мгновенного обновления статусов и событий звонков.

## Основные возможности
- **Real-time события:** Входящие звонки, изменение статуса очереди.
- **Управление статусом оператора:** (Online, Offline, Pause).
- **Мониторинг:** Отслеживание активных звонков.

## Компоненты
- `ContactCenterGateway` (`contact-center.gateway.ts`): WebSocket шлюз для связи с фронтендом.
- `ContactCenterService`: Логика распределения и управления состоянием.

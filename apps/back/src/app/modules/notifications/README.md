# Модуль Уведомлений (Notifications Module)

## Описание
Модуль отвечает за отправку и управление уведомлениями в системе. Поддерживает различные каналы доставки (In-app, Email, Push).

## Функциональность
- Создание уведомлений для пользователей.
- Отметка уведомлений как прочитанных.
- Рассылка системных событий.

## Компоненты
- `NotificationModule`: Настройка провайдеров уведомлений.
- `Controllers`: API для получения списка уведомлений пользователем.

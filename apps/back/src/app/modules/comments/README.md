# Модуль Комментариев (Comments Module)

## Описание
Универсальный модуль для добавления комментариев к различным сущностям системы (сделки, лиды, задачи).

## Функциональность
- Добавление текстовых комментариев.
- Поддержка вложений (опционально).
- История переписки.

## Компоненты
- `CommentsService`: Логика сохранения и выборки комментариев.
- `CommentsController`: API endpoints.

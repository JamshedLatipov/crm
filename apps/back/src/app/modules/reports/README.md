# Модуль Отчетов (Reports Module)

## Описание
Модуль для генерации аналитических отчетов по различным аспектам работы CRM (продажи, звонки, активность сотрудников).

## Возможности
- Агрегация данных за выбранный период.
- Отчеты по KPI менеджеров.
- Статистика по лидам и сделкам.

## Компоненты
- `ReportsService`: Сбор и обработка данных из других модулей/базы данных.
- `ReportsController`: API для выгрузки отчетов (JSON/CSV).

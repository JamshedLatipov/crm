# Модуль Скриптов Звонков (Call Scripts)

## Описание
Модуль управления сценариями разговоров (скриптами) для операторов колл-центра.

## Функциональность
- Создание деревьев скриптов (вопросы -> варианты ответов -> следующие вопросы).
- Категоризация скриптов.
- Отображение скрипта оператору во время звонка.

## Компоненты
- `CallScriptsService`: CRUD операции для скриптов.
- `CallScriptCategoryService`: Управление категориями.

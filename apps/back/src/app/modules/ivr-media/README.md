# Модуль IVR Media

## Описание
Модуль управляет медиа-файлами (аудиозаписями), используемыми в голосовых меню IVR и других голосовых сервисах.

## Функциональность
- Загрузка аудиофайлов (WAV, MP3).
- Конвертация форматов для Asterisk (при необходимости).
- Хранение и выдача файлов.

## Компоненты
- `IvrMediaService`: Логика загрузки и хранения.
- `IvrMediaController`: API для управления файлами.

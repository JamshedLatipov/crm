# Модуль Контактов (Contacts Module)

## Описание
Модуль отвечает за управление базой физических лиц (контактов) в CRM системе. Он позволяет создавать, редактировать, удалять и просматривать информацию о клиентах, а также отслеживать историю взаимодействия с ними.

## Основные компоненты

### Сущности (Entities)
- `ContactEntity` (`contact.entity.ts`): Основная сущность контакта. Содержит поля: имя, фамилия, телефон, email, должность и др.
- `ContactActivityEntity` (`contact-activity.entity.ts`): История действий, связанных с контактом (звонки, встречи, изменения).

### Сервисы (Services)
- `ContactsService` (`contacts.service.ts`): Реализует бизнес-логику работы с контактами.
  - CRUD операции.
  - Поиск и фильтрация.
  - Связывание с компаниями и сделками.

### Контроллеры (Controllers)
- `ContactsController` (`contacts.controller.ts`): Предоставляет REST API для работы с контактами.

## API Endpoints

- `GET /api/contacts` - Получение списка контактов (с пагинацией и фильтрацией).
- `GET /api/contacts/:id` - Получение детальной информации о контакте.
- `POST /api/contacts` - Создание нового контакта.
- `PATCH /api/contacts/:id` - Обновление информации о контакте.
- `DELETE /api/contacts/:id` - Удаление контакта.

# Модуль Пользователей (User Module)

## Описание
Модуль управляет пользователями системы (сотрудниками и администраторами), аутентификацией и авторизацией.

## Основные возможности
- **Аутентификация:** JWT (JSON Web Tokens).
- **Авторизация:** Ролевая модель (RBAC) с использованием декораторов `@Roles`.
- **Управление пользователями:** CRUD операции, блокировка, смена пароля.

## Компоненты

### Сервисы
- `UserService`: Управление данными пользователей.
- `AuthService`: Логика входа, регистрации и выдачи токенов.

### Guards (Защитники)
- `JwtAuthGuard`: Проверка валидности токена.
- `RolesGuard`: Проверка прав доступа.

### Декораторы
- `@CurrentUser()`: Получение текущего пользователя из запроса.

## API Endpoints
- `POST /api/auth/login` - Вход в систему.
- `POST /api/auth/register` - Регистрация (если открыта).
- `GET /api/user/profile` - Получение профиля текущего пользователя.
- `GET /api/user` - Список пользователей (для админов).

# Модуль IVR (Interactive Voice Response)

## Описание
Модуль отвечает за логику и выполнение голосовых меню (IVR). Позволяет настраивать маршрутизацию звонков, воспроизведение приветствий и обработку ввода пользователя (DTMF).

## Основные компоненты

### Сервисы
- `IvrService` (`ivr.service.ts`): Управление конфигурациями IVR (создание, редактирование меню).
- `IvrRuntimeService` (`ivr-runtime.service.ts`): Исполнение логики IVR в реальном времени при входящем звонке.

### API
- `IvrController`: Эндпоинты для настройки схем IVR.

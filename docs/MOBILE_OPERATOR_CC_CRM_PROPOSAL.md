# КОММЕРЧЕСКОЕ ПРЕДЛОЖЕНИЕ  
## Контакт-центр + CRM для сотового оператора

### Ключевая ценность
- Рост NPS и удержания за счет быстрого, предсказуемого сервиса.
- Больше продаж VAS/доп.опций через точный апсейл.
- Ниже стоимость минуты поддержки и быстрее обучение операторов.

### Платформа и модули
- Единое решение КЦ + CRM (NestJS + Angular, Asterisk/JsSIP, RabbitMQ, Redis, Postgres).
- Модули: омниканальные очереди/кампании, WebRTC софтфон, IVR, маршрутизация по навыкам, карточка абонента, тикеты/заявки, продажи/апсейл, аналитика в реальном времени.
- Быстрые доработки под BSS/OSS: тарифы, балансы, блокировки, SIM/ESIM, портирование, покрытие/качество связи, SLA для корп-сегмента.

### Контакт-центр
- WebRTC софтфон, статус Registered! после звонков, запись/расшифровка, опциональное шумоподавление.
- Маршрутизация: навыки, VIP/корп, гео/регион, продукт, время ожидания; fallback по правилам.
- IVR/меню, callback по расписанию, кампании обзвона (progressive/predictive/manual), blended-режимы.
- Контроль качества: скрипты, чек-листы, автоалерты по ключевым словам, оценки звонков.
- Статусы операторов, перерывы, мониторинг в реальном времени, экспорт событий в BI.

### CRM
- Карточка абонента: услуги, тариф, баланс, долги, заявки, жалобы, устройства, SIM/ESIM, договоры/аккаунты.
- Продажи и апсейл: рекомендации по сегменту/ARPU/качеству связи, корзина опций, KYC/AML чеклист, интеграция с платежами/эквайрингом.
- Заявки и инциденты: шаблоны, SLA-таймеры, эскалации, маршруты по роли и продукту.
- Корпоративные клиенты: аккаунты, договоры, пулы номеров, приоритетные очереди, выделенные SLA.
- Коммуникации: SMS/WhatsApp/Email/Push, шаблоны, переменные, персонализация.

### Интеграции
- BSS/CRM ядро: тарифы, баланс, блокировки, детализация, платежи.
- OSS: качество сети, события деградаций, плановые работы.
- NPI/портирование, проверки номера, HLR/LNP.
- BI/DWH: события звонков, заявки, продажи в RabbitMQ-шину или прямые выгрузки.

### Безопасность
- RBAC/ABAC, аудит действий, шифрование WebRTC и API, ограничение доступа к записям.
- Разграничение персональных данных по ролям (операторы, супервайзеры, аутсорс).
- Логи и метрики: централизованный сбор, алерты на деградацию (SIP, RabbitMQ, Postgres).

### План внедрения (оценка)
- Неделя 1: обследование, интеграционные потоки BSS/OSS, карта процессов и SLA.
- Недели 2–3: пилот КЦ (входящие очереди, IVR, WebRTC, статусы, базовая аналитика).
- Недели 3–4: CRM карточка абонента, заявки/тикеты, биллинг-интеграция, шаблоны уведомлений.
- Недели 5–6: кампании обзвона, апсейл/продажи, корпоративные очереди, QA/скрипты.
- Недели 7–8: нагрузочные тесты, обучение, запуск в прод, поддержка первой линии.

### Лицензирование (варианты)
- Подписка за оператора/месяц (КЦ + CRM + софтфон).
- Пакет минут/кампаний для обзвона + SaaS за функционал CRM.
- On-prem с годовой поддержкой и SLA.

### SLA и поддержка
- Uptime: 99.5% (стандарт) / 99.9% (расширенный) при кластерной схеме.
- Support: рабочие часы/24x7, реакции P1 < 30 мин, P2 < 2 ч, P3 < рабочий день.
- Обновления: ежемесячные миноры, ежеквартальные мажоры.

### Почему мы
- Собственный стек Asterisk/JsSIP и готовый софтфон без плагинов.
- Быстрые кастомизации под процессы оператора, короткий TTM благодаря готовым модулям Nx монорепо.
- Опыт интеграций с биллингами, NPI/LNP, платежами и сегментными кампаниями.

### Следующие шаги
- Созвон для уточнения процессов (поддержка, продажи, корпоративный сегмент) и KPI.
- Демонстрация пилота: очереди, маршрутизация, карточка абонента, апсейл.
- Совместное утверждение дорожной карты и модели лицензирования.
